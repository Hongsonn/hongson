# Cài ssl free Let's Encrypt 

1. Kiểm tra xem Hostname đã cài đặt ssl chưa tại sslshopper.com

![a](https://f6-zpcloud.zdn.vn/3023278266996304687/3944cd093f78f326aa69.jpg)

2. Truy cập Kerio Connect

Click lần lượt: `Configuration => SSL Certificates => New => New Let's Encrypt Certificate`

![a](https://f6-zpcloud.zdn.vn/370473169778570167/db7c7b33ba42761c2f53.jpg)

3. Nhập Hostname muốn cài dặt.

![a](https://f6-zpcloud.zdn.vn/1341060648220099214/26412915dc64103a4975.jpg)

4. Chứng chỉ SSL sẽ được cài đặt tự động, không cần thao tác gì thêm

5. Cài đặt thành công 

![a](https://f5-zpcloud.zdn.vn/2843814146345108959/ebbba253bf23737d2a32.jpg)
